    About

    Built by Sanjay

    Engineering student & indie developer

    Why I Built This

    I built this Chrome new-tab extension to create a calmer and more intentional start to every browsing session. Most existing dashboards felt either cluttered or too limited in customization. I wanted a balance — something visually clean, flexible, and genuinely useful for everyday focus.

    This project started as a personal tool and evolved into a fully customizable dashboard with movable widgets, live wallpapers, and fine-grained layout control. It reflects my interest in building practical, well-designed products that people can shape around their own workflow.

    Built using modern web technologies with a focus on performance, responsiveness, and clean UI design.

    How to use

    ×

    Quick tips to get started. Press Esc anytime to close.

    Search

    Type a query or a URL in the top bar and press Enter.

    Customize layout

    Click the 🔒 button to enter Customize mode.

    • Move: drag by the handle
    • Hide: toggle in Settings
    • Reset: use Reset layout in Settings

    Background

    Click ⚙️ to change the background image (upload or paste an image URL).

    Wallpaper Zoom

    Zoom up to 250% in Settings. Use pan sliders to adjust the view and create a custom arrangement.

    Responsive behavior

    When the window becomes narrow, widgets automatically stack. On a wide window, your custom layout is restored.

    Shortcuts

    Esc — close dialogs / return to dashboard

    Enter — search / open first item
